# ToDo List


## Задача: 

    Создание ToDo листа c возможностью добавления новых задач и их удаления.

---

## Запуск:

    1. Клонируйте репозиторий

    > git clone

    2. Скачайте использованные в проекте пакеты

    > npm install

    3. Запустите транспилляцию 

    > npm run watch
